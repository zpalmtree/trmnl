## Quick Reference

- **Stack**: Cloudflare Workers, TypeScript
- **API**: Cinder backend at sol-incinerator.dev/api (requires API key)
- **Deploy**: `npm run deploy`
- **Test**: `npm run dev` then visit http://localhost:8787
- **Set secret**: `npx wrangler secret put CINDER_API_KEY`
